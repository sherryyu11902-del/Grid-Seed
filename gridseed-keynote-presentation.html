初衷与命题 · Intent

GridSeed exists so every designer—especially beginners—can reach professional layout in Illustrator without hours of manual scaffolding. The intent is a guided, repeatable workflow instead of blank-artboard anxiety.

让新手也能在 Illustrator 内快速获得「专业感」版式骨架：用可复现的流程，替代从零开始画参考线、画框与零散样式。

要解决什么问题 · Problems

  • Manual guides, frames, borders, shadows, and local styles built block-by-block waste time and break rhythm.逐块手搭参考线、画框、描边与阴影，节奏慢且易乱。
  • Beginners stall between “looking pro” and shipping; veterans repeat the same setup on every artboard.新手卡在「显得专业」与「按时交稿」之间；熟手反复做相同前置工序。
  • Piecemeal spacing hurts brand consistency; print workflows need bleed, trim, and safe zones—often forgotten under pressure.零碎作业导致间距与风格不一；印刷交付易漏出血、裁切与安全区。

现有能力一览 · Capabilities

Grid Composer — themes, paper/screen formats, live preview with true aspect ratio, layout presets (golden, modular, masonry, columns), image slots with scale/gap, print-ready bleed & safe margins, placement and optional PNG preview export.

Templates & Starter Kit — format chips (social, poster, editorial, custom pixels), grid levels L1–L6, cyan construction guides, trim/safe guides when bleed > 0, curated Starter Kit artwork that scales with the artboard.

模板 / 网格编排器 / Starter Kit 三者闭环：从选幅面、选版式，到一键置入可编辑的构建线与成品化基础图层。

详细功能 · Deep dives（对照幻灯片 5–11、19–21）

  • Orange frames mark image slots; dashed guides mirror safe layout logic from the panel. 橙色线框为图片占位；虚线区对应面板内可排版安全区逻辑。
  • Starter Kit themes bundle color, shape language, and type stacks—not flat gray placeholders. Starter Kit 主题整合配色、图形语汇与字体栈——不是灰色空框。
  • Cyan grids encode column/module rhythm; lock or hide after client sign-off. 青色网格定义栏与模块节奏；定稿后可锁定或隐藏构建层。
  • Print mode: magenta bleed, orange trim, cyan type safe zones when bleed is enabled. 印刷模式：出血为洋红示意、裁切为橙色、正文安全区为青色（启用出血时）。
  • Architecture split: CEP HTML/JS panel ↔ JSX host in Illustrator for fast UI iteration. 架构：CEP 面板（HTML/JS）与 Illustrator JSX 宿主分离，便于界面快速迭代。

如何使用 · Step-by-step（对照幻灯片 16）

  1. Install & open the CEP extension via Window ▸ Extensions ▸ GridSeed.安装扩展，必要时重启 Illustrator，从「窗口 ▸ 扩展」打开面板。
  2. Choose a path — Templates tab for chips & levels, or Grid Composer for full guided builds.选「模板」快速幅面，或「网格编排器」完整流程。
  3. Dial in settings — orientation, grid type, image slots, scale/gap; add bleed/safe for print.设置横竖、网格、图片槽与缩放/间距；印刷任务填写出血与安全边距。
  4. Preview live in-panel; export PNG for review before placing vectors.预览可导出 PNG 供评审。
  5. Create in Illustrator — Place / Regenerate: guides, Starter Kit, and frames land on the artboard.置入后，参考线、Starter Kit 与占位一同上画板。
  6. Customize & ship — swap imagery and type; export print PDFs or social assets.在结构内替换内容，再按规格导出。

演示故事线 · Demos for novice designers（对照幻灯片 17–18）

  • Mood board sprint: preset first → theme second → place once → finish by swapping imagery while the grid enforces rhythm.氛围板：先预设、再主题、一键置入，替换图片仍保持精致比例。
  • Instagram / portfolio strips: match canvas in preview, pick modular/editorial rhythm, tweak scale/gap without redrawing frames, duplicate the recipe for carousel pages.社交轮播 / 作品集条带：预览与画板一致，多页复制同一套间距规则。

四条关键信息 · Key messages

Speed · 速度 Instant professional grids replace slow, pixel-by-pixel manual frames, borders, shadows, and one-off styles. 专业网格瞬间就绪，取代逐一手动画框、描边、阴影与一次性样式。
Quality · 质量 Consistent, polished outcomes because every scaffold ships with curated Starter Kit typography, color, and shapes. 每套网格搭配精心打磨的 Starter Kit，字体、色彩与图形一致且耐看。
Versatility · 灵活 One plugin for mood boards, photo collages, portfolio spreads, Instagram carousels, product grids, lookbooks, editorial modules, and pitch decks. 同一插件适用于氛围板、拼贴、作品集、社交轮播、产品格、造型册、编辑版与提案等。
Transformation · 转变 Turns traditional slow, repetitive setup into a fast, confident, high-quality workflow—from grind to design. 把缓慢、重复的前置工序变为快速、省力且高质量的流程。
